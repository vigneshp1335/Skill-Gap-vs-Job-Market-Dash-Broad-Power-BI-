# Skill Gap vs Job Market Dashboard
## Project Overview
The **Skill Gap vs Job Market Dashboard** is an interactive data analytics project designed to analyze the mismatch between candidate skills and job market demand.
This dashboard provides insights into hiring trends, skill demand levels, industry job distribution, and salary patterns.
## Tools & Technologies Used
- Power BI / Excel (Update with your tool)
- Data Cleaning & Transformation
- Data Modeling
- Interactive Dashboard Design
- DAX
## Dashboard Features
### Interactive Filters
- Industry
- Year
- Location
- Years of Experience
### Key Performance Indicators (KPIs)
- 500 Total Candidates
- 6 Skill Categories
- 500 Total Job Postings
- 909.14K Average Salary
## Visualizations Included (Donut Chart)
### Candidate Hiring Status
- Donut chart showing Hired vs Not Hired candidates.
### Skill Category Demand Analysis (Bar Chart)
- Programming & Analytics → High demand
- Communication & Marketing → Moderate demand
### Hiring Trend by Experience (Line Chart)
- Analyzes how hiring varies based on years of experience.
