# Skill-Gap-vs-Job-Market-Dash-Broad-Power-BI
Project Overview (You Can Paste This in GitHub)

Skill Gap vs Job Market Dashboard is an interactive data visualization project built to analyze the mismatch between candidate skills and job market demand.

The dashboard helps identify:

Which skills are most in demand

Hiring trends across industries

Skill category distribution

Salary insights

Candidate hiring status

Industry-wise job demand

This project demonstrates how data analytics can be used to understand employment trends and workforce skill gaps.
